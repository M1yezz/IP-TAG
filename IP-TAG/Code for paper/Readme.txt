# Readme

# The processed data used in the paper are in the file "dataset" .

# Running "IP-TAG.ipynb" can obtain the positioning error of the IP-TAG, which is the proposed positioning method of the paper.

# Running other ".ipynb" can obtain the positioning error of other positioning method, which can be used for comparison with IP-TAG.

# Running "Show result (CDF).ipynb" can obtain the cumulative distribution function (CDF) of the positioning errors.
